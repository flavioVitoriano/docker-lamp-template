## Template docker php

esse template foi feito visando facilitar o uso de um servidor LAMP em docker.

### serviços
* web: Dockerfile - local
* db: mysql
* phpmyadmin: phpmyadmin


### rodar
para rodar o projeto, rode:
    ```
    docker-compose up
    ```
